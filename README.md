# バンディットアルゴリズム

[Qiita](https://qiita.com/birdwatcher)で紹介したバンディットの実装。

## Bandit
### 0/1報酬
```sh
python -m binary
```

### 連続報酬
```sh
python -m continuous
```

## Contextual Bandit
### 0/1報酬
```sh
python -m contextual_binary
```
```sh
python -m contextual_binary_cluster
```

### 連続報酬
```sh
python -m contextual_continuous
```

## Cascading Bandit
```sh
python -m cascading
```

## Contextual Cascading Bandit
```sh
python -m contextual_cascading
```
